Политика обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных (далее — «Политика») определяет порядок и условия обработки данных, предпринимаемые платформой Radon Scope (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этой информации.

1.1. Важнейшей целью Оператора является соблюдение прав и свобод человека при обработке его персональных данных, включая защиту прав на неприкосновенность частной жизни и профессиональную тайну в рамках карьерного консультирования.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях веб-сайта radon-scope.blog.

2. Основные понятия

  • Веб-сайт — совокупность графических и информационных материалов Radon Scope.
  • Пользователь — любой посетитель сайта.
  • Персональные данные — любая информация, относящаяся к определенному Пользователю (имя, email, телефон).
  • Обработка данных — любые действия с данными (сбор, хранение, уточнение, удаление).

3. Данные, которые мы обрабатываем

Оператор может обрабатывать следующие категории данных:

  • Предоставляемые Пользователем: Имя, адрес электронной почты, номер телефона (для связи с экспертом).
  • Собираемые автоматически: Обезличенные данные о посетителях (файлы «cookie») с помощью сервисов Google Аналитика для улучшения структуры блога.

4. Цели обработки данных

  • Идентификация Пользователя для предоставления доступа к результатам теста и аналитике.
  • Установление обратной связи для проведения карьерных консультаций.
  • Информирование о новых стратегиях роста и AI-инструментах (с согласия Пользователя).

5. Правовые основания

Оператор обрабатывает данные только при наличии добровольного согласия Пользователя, выраженного путем заполнения форм на сайте radon-scope.blog или использования интерактивных элементов (квиз, радар инноваций).

6. Безопасность и хранение

Оператор принимает все необходимые технические меры для защиты данных от неправомерного доступа. Данные Пользователя никогда не будут переданы третьим лицам, за исключением случаев, предусмотренных законодательством ЕС. Срок обработки данных является неограниченным, но Пользователь может в любой момент отозвать согласие.
